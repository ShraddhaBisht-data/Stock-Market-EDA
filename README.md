# Stock Market EDA 📈

## Project Overview
This project performs Exploratory Data Analysis (EDA) on NASDAQ listed securities data.

The objective is to analyze:
- Exchange distribution
- ETF listings
- Financial status
- Security categories
- Symbol patterns

## EDA Process
1. Data Cleaning
2. Missing Value Handling
3. Univariate Analysis
4. Bivariate Analysis
5. Multivariate Analysis
6. Feature Engineering

## Key Insights
- NASDAQ contains a large concentration of non-ETF securities.
- Short ticker symbols dominate the exchange listings.
- Certain exchanges contain higher ETF representation.
- Financial status categories show imbalance across listings.
